  • Oleh : Dian Ramadhani

Setiap mahasiswa mempunya kewajiban mempublikasi hasil penelitian skripsi dalam bentuk poster. Berikut daftar poster hasil skripsi mahasiswa Program Studi Teknik Informatika, Jurusan Teknik Elektro, Fakultas Teknik, Universitas Riau.

  1. JUDUL SKRIPSI : Perbandingan Manajemen Bandwidth Metode Hierarchical Token Bucket (HTB) Dan Peer Connection Queue (PCQ) Pada SDS IT Sahabat Muslim Duri
  2. JUDUL SKRIPSI : Rancang Bangun Sistem Diagnosa Penyakit Paru-Paru Menggunakan Metode Metode Naive Bayes Berbasis Website
  3. JUDUL SKRIPSI : Implementasi Deep Learning Untuk Identifikasi Daun Tanaman Obat Menggunakan Metode Transfer Learning
  4. Judul : Sistem Keamanan Sepeda Motor Menggunakan Teknik Pelacakan Berbasis Internet of Things (IoT) Dengan Komunikasi Long Range (LoRa)
  5. Judul : Analisis Kerentanan dan Keamanan Layanan Cloud Computing dari Serangan Denial of Service (DoS) dengan Menggunakan Metode Intusion Detection System (IDS).
